# 记忆宅邸：失忆管家 - 关卡设计

## 关卡总览

游戏包含四个关卡，串联成一天的故事线：

| 关卡 | 名称 | 时段 | 难度 | 记忆类型 | 核心NPC |
|------|------|------|------|---------|---------|
| 1 | 初次整理 | 🍽️ 第一天（教学） | tutorial | 空间记忆 + 物体记忆 | MEM-07 |
| 2 | 出门大作战 | 🌅 清晨 8:00 | medium | 空间记忆 + 物体位置 | 钥匙猫 |
| 3 | 洗衣幽灵 | 👻 下午 15:00 | medium | 计数记忆 + 程序记忆 | 袜子幽灵 |
| 4 | 深夜巡逻 | 🌙 深夜 02:00 | medium-hard | 空间记忆 + 时间记忆 | MEM-07（巡逻模式） |

## 关卡 1：初次整理（教学关卡）

### 1.1 任务信息

- **ID**: `task-clean-table`
- **名称**: 初次整理
- **时间限制**: 180秒（3分钟）
- **难度**: tutorial（新手教学）
- **记忆类型**: 空间记忆入门、物体记忆、程序记忆
- **房间**: 餐厅

### 1.2 任务背景

MEM-07 的记忆模块出了故障，只能同时记住 3 件物品。从简单的整理任务开始——把餐桌上的脏杯子、餐巾纸和叉子归位。

### 1.3 设计目标

- 让玩家学会基本操作（移动、拾取、放置、保存记忆）
- 建立有限记忆槽的概念
- 熟悉目标区标记系统
- 无压力入门体验

### 1.4 目标物品

| 物品 | 初始位置 | 目标位置 | 描述 |
|------|---------|---------|------|
| 脏杯子 | 餐桌左侧 | 洗碗机 | 需要清洗的灰色杯子 |
| 餐巾纸 | 餐桌右侧 | 垃圾桶 | 用过的黄色餐巾纸 |
| 叉子 | 餐桌中央前方 | 餐具架 | 需要归位的银色叉子 |

### 1.5 目标容器

| 容器 | 位置 | 接受物品 | 视觉标记 |
|------|------|---------|---------|
| 洗碗机 | 餐厅右侧 | cup | 蓝色发光标记 |
| 垃圾桶 | 餐厅左侧 | tissue | 红色发光标记 |
| 餐具架 | 餐厅中间左侧 | fork | 黄色发光标记 |
| 餐桌 | 餐厅中央 | - | 初始物品所在 |

### 1.6 分步引导

| 步骤 | 引导内容 | 类型 |
|------|---------|------|
| 1 | 欢迎提示，介绍桌上三件物品 | info |
| 2 | 靠近物品按 F 拾取 | info |
| 3 | 按 E 保存记忆，记忆槽有限 | info |
| 4 | 拿着物品靠近目标区按 F 放置 | info |
| 6 | 鼓励提示 | info |
| 8 | 提示洗碗机位置（蓝色标记） | info |
| 10 | 提示垃圾桶位置（红色标记） | info |
| 12 | 提示餐具架位置（黄色标记） | info |

### 1.7 记忆测试 (Probes)

| 类型 | 问题 | 选项 |
|------|------|------|
| 位置 | 脏杯子一开始放在哪里？ | 餐桌上 / 洗碗机里 / 垃圾桶里 |
| 位置 | 餐巾纸应该扔到哪里？ | 洗碗机 / 垃圾桶 / 餐桌 |
| 位置 | 叉子应该放到哪里？ | 洗碗机 / 垃圾桶 / 餐具架 |

## 关卡 2：出门大作战

### 2.1 任务信息

- **ID**: `task-leave-home`
- **名称**: 出门大作战
- **时间限制**: 240秒（4分钟）
- **难度**: medium（中等）
- **记忆类型**: 空间记忆、物体位置记忆
- **房间**: 客厅、卧室、厨房、玄关

### 2.2 任务背景

早上八点，主人要出门上班。钥匙猫调皮地把钥匙和钱包扒拉得到处都是。玩家需要找到钥匙、手机、雨伞和钱包，放到玄关托盘上。

### 2.3 目标物品

| 物品 | 初始位置 | 目标位置 | 特殊说明 |
|------|---------|---------|---------|
| 🔑 钥匙 | 客厅茶几 | 玄关托盘 | 钥匙猫会移动它 |
| 📱 手机 | 卧室床头柜抽屉 | 玄关托盘 | 需要打开抽屉 |
| ☂️ 雨伞 | 玄关伞架 | 玄关托盘/伞架 | 已在目标区域附近 |
| 👛 钱包 | 客厅沙发 | 玄关托盘 | 钥匙猫会移动它 |

### 2.4 目标容器

| 容器 | 位置 | 接受物品 |
|------|------|---------|
| 玄关托盘 | 玄关 | 钥匙、手机、雨伞、钱包 |
| 茶几 | 客厅 | - |
| 床头柜抽屉 | 卧室 | - |
| 伞架 | 玄关 | 雨伞 |
| 主沙发 | 客厅 | - |

### 2.5 捣乱事件

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 钥匙猫推钥匙 | 第8步后 | 钥匙移动到沙发旁边 |
| 钥匙猫藏钱包 | 第12步后 | 钱包移动到沙发缝 |
| 手机响铃 | 第3步 | 提示手机在卧室抽屉 |
| 主人催促 | 第5步 | 紧迫感提示 |
| 钱包提示 | 第15步 | 提示钱包在沙发附近 |

### 2.6 记忆测试 (Probes)

| 类型 | 问题 | 选项 |
|------|------|------|
| 位置 | 钥匙最初放在哪里？ | 客厅茶几上 / 卧室抽屉 / 厨房台面 / 玄关伞架 |
| 位置 | 手机一开始放在哪里？ | 客厅茶几 / 卧室抽屉 / 厨房台面 / 玄关托盘 |
| 状态 | 手机最初是否可见？ | 可见 / 不可见（藏在抽屉） |

## 关卡 4：深夜巡逻

### 4.1 任务信息

- **ID**: `task-night-patrol`
- **名称**: 深夜巡逻
- **时间限制**: 300秒（5分钟）
- **难度**: medium-hard（中高难度）
- **记忆类型**: 空间记忆、时间记忆、物体记忆
- **房间**: 客厅、卧室、厨房、玄关、餐厅

### 4.2 任务背景

深夜两点，主人已经熟睡。MEM-07 启动夜间巡逻模式——黑暗中似乎有东西被挪动了，还有电器异响和窗户晃动。玩家需要在黑暗中巡查所有房间，找到被打乱的物品并确认归位。

### 4.3 核心玩法

- 视野受限（黑暗中只能看清眼前物体）
- 屏幕边缘方向指示标记待确认物品位置
- 夜间扰动事件会移动物品位置
- 需要依次巡查所有房间

## 关卡设计原则

### 难度递进

```
关卡1（教学）→ 基础操作学习，无捣乱事件，3件物品，180秒
关卡2（出门）→ 空间记忆，钥匙猫捣乱，4件物品，240秒
关卡3（洗衣）→ 分类记忆，袜子幽灵捣乱，9件物品，120秒（限时挑战）
关卡4（巡逻）→ 黑暗环境，夜间扰动，4件物品，300秒（视野受限）
```

### 记忆类型覆盖

每个关卡侧重不同的记忆类型，全面训练玩家的记忆能力：

| 记忆类型 | 训练关卡 | 核心挑战 |
|---------|---------|---------|
| 空间记忆 | 所有关卡 | 记住物品位置 |
| 物体记忆 | 关卡1、2、3 | 识别物品属性和状态 |
| 计数记忆 | 关卡3 | 记住物品数量 |
| 程序记忆 | 关卡1、3 | 记住操作序列和规则 |
| 时间记忆 | 关卡4 | 记住时间变化和事件 |

### 叙事连贯性

四个关卡串联成完整的一天故事线：

1. **初次整理** → MEM-07 自我介绍，说明记忆故障
2. **出门大作战** → 钥匙猫捣乱，主人出门上班
3. **洗衣幽灵** → 下午洗衣任务，袜子幽灵捣乱
4. **深夜巡逻** → 深夜巡查，夜间扰动事件

### 房间物品合理性

每个房间的物品都符合现实生活逻辑：

| 房间 | 合理物品 |
|------|---------|
| 餐厅 | 桌子、椅子、餐具、垃圾桶、洗碗机 |
| 客厅 | 茶几、沙发、电视、遥控器 |
| 卧室 | 床、床头柜、抽屉、手机 |
| 玄关 | 托盘、伞架、钥匙、雨伞 |
| 洗衣房 | 洗衣机、洗衣篮、衣物、毛巾 |
| 厨房 | 冰箱、橱柜、台面、碗 |
