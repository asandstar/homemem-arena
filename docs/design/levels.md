# HomeMem Arena - 关卡设计

## 关卡总览

游戏包含四个关卡，串联成一天的故事线：

| 关卡 | 名称 | 时段 | 难度 | 记忆类型 |
|------|------|------|------|---------|
| 1 | 出门大作战 | 🌅 清晨 8:00 | 中等 | 空间记忆 + 物体位置 |
| 2 | 餐桌混乱 | 🍽️ 上午 9:30 | 简单 | 物体状态 + 类别规则 |
| 3 | 袜子分拣 | 👕 下午 14:00 | 困难 | 配对记忆 + 程序记忆 |
| 4 | 早餐准备 | ☕ 深夜（时间循环） | 困难 | 时间记忆 + 空间记忆 |

---

## 关卡 1：出门大作战

### 1.1 任务信息

- **ID**: `task-leave-home`
- **名称**: 出门大作战
- **时间限制**: 10 分钟（600秒）
- **难度**: 中等
- **记忆类型**: 空间记忆、物体位置记忆

### 1.2 任务背景

早上八点，主人要出门上班。钥匙猫调皮地把钥匙扒拉得到处都是。玩家需要找到钥匙、手机和雨伞，放到玄关托盘上。

### 1.3 目标物品

| 物品 | 初始位置 | 目标位置 | 特殊说明 |
|------|---------|---------|---------|
| 🔑 钥匙 | 客厅茶几 | 玄关托盘 | 钥匙猫会移动它 |
| 📱 手机 | 卧室床头柜抽屉 | 玄关托盘 | 需要打开抽屉 |
| ☂️ 雨伞 | 玄关伞架 | 玄关托盘/伞架 | 已在目标区域附近 |

### 1.4 目标容器

| 容器 | 位置 | 接受物品 |
|------|------|---------|
| 玄关托盘 | 玄关 | 钥匙、手机、雨伞 |
| 茶几 | 客厅 | - |
| 床头柜抽屉 | 卧室 | - |
| 伞架 | 玄关 | 雨伞 |

### 1.5 捣乱事件

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 钥匙猫推钥匙 | 第4步后 | 钥匙移动到客厅角落 |
| 钥匙猫叼钥匙 | 第10步后 | 钥匙移动到卧室 |
| 手机响铃 | 第2步 | 提示手机在卧室抽屉 |
| 主人催促 | 第3步 | 紧迫感提示 |

### 1.6 记忆测试 (Probes)

| 类型 | 问题 | 选项 |
|------|------|------|
| 位置 | 钥匙最初放在哪里？ | 客厅茶几上 / 卧室抽屉 / 厨房台面 / 玄关伞架 |
| 位置 | 手机一开始放在哪里？ | 客厅茶几 / 卧室抽屉 / 厨房台面 / 玄关托盘 |
| 状态 | 手机最初是否可见？ | 可见 / 不可见（藏在抽屉） |
| 计数 | 需要带到玄关的物品一共有几件？ | 2 / 3 / 4 / 5 |

## 关卡 2：餐桌混乱

### 2.1 任务信息

- **ID**: `task-clean-table`
- **名称**: 餐桌混乱
- **时间限制**: 75秒
- **难度**: 简单
- **记忆类型**: 物体状态记忆、类别规则记忆、空间记忆

### 2.2 任务背景

早餐后留下一桌子狼藉，调皮的餐盘精总爱把脏盘子偷偷放回桌上！玩家需要快速识别并归类物品。

### 2.3 目标物品

| 物品 | 类别 | 状态 | 目标位置 |
|------|------|------|---------|
| 脏杯子 | cup | dirty | 洗碗机/水槽 |
| 干净杯子 | cup | clean | 不扔进垃圾桶即可 |
| 脏盘子 | plate | dirty | 洗碗机/水槽 |
| 餐巾纸 | tissue | - | 垃圾桶 |
| 遥控器 | remote | - | 客厅茶几 |

### 2.4 目标容器

| 容器 | 位置 | 接受物品 |
|------|------|---------|
| 洗碗机 | 厨房 | cup, plate |
| 水槽 | 厨房 | cup, plate |
| 垃圾桶 | 厨房 | tissue |
| 客厅茶几 | 客厅 | remote |

### 2.5 捣乱事件

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 餐盘精放回盘子 | 第8步 | 脏盘子回到餐桌 |
| 餐盘精放回杯子 | 第12步 | 脏杯子回到餐桌 |
| 相似杯子提示 | 第3步 | 系统提醒区分颜色 |
| 室友消息 | 第4步 | 提醒注意区分杯子 |

### 2.6 记忆测试 (Probes)

| 类型 | 问题 | 选项 |
|------|------|------|
| 计数 | 餐桌上一共有几个杯子？ | 1 / 2 / 3 / 4 |
| 物体识别 | 哪个杯子需要清洗？ | 白色亮杯 / 灰色杯子 |
| 位置 | 餐巾纸应该扔到哪个容器？ | 洗碗机 / 水槽 / 垃圾桶 |
| 位置 | 遥控器应该放到哪里？ | 洗碗机 / 水槽 / 垃圾桶 / 客厅茶几 |

## 关卡 3：袜子分拣

### 3.1 任务信息

- **ID**: `task-laundry-sort`
- **名称**: 袜子分拣
- **难度**: 困难
- **记忆类型**: 配对记忆、程序记忆

### 3.2 任务背景

洗衣篮里堆满了待分拣的袜子。袜子幽灵会交换篮子里的袜子，考验玩家的配对记忆能力。

### 3.3 核心玩法

- 识别成对的袜子（颜色、图案匹配）
- 将配对的袜子放入正确的抽屉
- 袜子幽灵会交换袜子位置
- 需要记住袜子的配对关系

---

## 关卡 4：早餐准备

### 4.1 任务信息

- **ID**: `task-breakfast`
- **名称**: 早餐准备
- **难度**: 困难
- **记忆类型**: 时间记忆、空间记忆

### 4.2 任务背景

时间循环发生，玩家需要在循环中记住物品位置变化，准备一顿完美的早餐。

### 4.3 核心玩法

- 收集食材准备早餐
- 注意时间循环带来的位置变化
- 在循环中优化策略

## 关卡设计原则

### 难度递进

```
关卡1 → 基础操作学习，单一捣乱者
关卡2 → 状态识别，多物品处理
关卡3 → 配对记忆，复杂规则
关卡4 → 时间循环，高阶策略
```

### 记忆类型覆盖

每个关卡侧重不同的记忆类型，全面训练玩家的记忆能力：

| 记忆类型 | 训练关卡 | 核心挑战 |
|---------|---------|---------|
| 空间记忆 | 关卡1、4 | 记住物品位置 |
| 物体记忆 | 关卡2 | 识别物品属性和状态 |
| 配对记忆 | 关卡3 | 记住物品配对关系 |
| 程序记忆 | 关卡2、3 | 记住操作序列和规则 |
| 时间记忆 | 关卡4 | 记住时间变化和循环 |

### 叙事连贯性

四个关卡串联成完整的一天故事线，每个关卡都有独立的叙事和角色互动。
