# 记忆宅邸：失忆管家 - 关卡设计

## 关卡总览

游戏包含四个关卡，串联成一天的故事线：

| 关卡 | 名称 | 时段 | 难度 | 记忆类型 | 核心NPC |
|------|------|------|------|---------|---------|
| 1 | 初次整理 | 🍽️ 第一天（教学） | 新手 | 空间记忆 + 物体记忆 | MEM-07 |
| 2 | 出门大作战 | 🌅 清晨 8:00 | 中等 | 空间记忆 + 物体位置 | 钥匙猫 |
| 3 | 餐桌混乱 | 🍽️ 上午 9:30 | 简单 | 物体状态 + 类别规则 | 餐盘精 |
| 4 | 洗衣幽灵 | 👕 下午 14:00 | 困难 | 配对记忆 + 程序记忆 | 袜子幽灵 |
| 5 | 早餐时间循环 | ⏰ 深夜（时间循环） | 困难 | 时间记忆 + 空间记忆 | 早餐闹钟 |

> **注意**：文档中列出5个关卡是为了保持叙事完整性，但实际代码实现中「餐桌混乱」已被重做为「初次整理」教学关卡。原设计中的第二关「出门大作战」变为实际第二关。

## 关卡 1：初次整理（教学关卡）

### 1.1 任务信息

- **ID**: `task-clean-table`
- **名称**: 初次整理
- **时间限制**: 300秒（5分钟）
- **难度**: tutorial（新手教学）
- **记忆类型**: 空间记忆入门、物体记忆
- **房间**: 餐厅

### 1.2 任务背景

MEM-07 的记忆模块出了故障，只能同时记住 3 件物品。从简单的整理任务开始——把餐桌上的脏杯子和餐巾纸归位。

### 1.3 设计目标

- 让玩家学会基本操作（移动、拾取、放置、保存记忆）
- 建立有限记忆槽的概念
- 熟悉目标区标记系统
- 无压力入门体验

### 1.4 目标物品

| 物品 | 初始位置 | 目标位置 | 描述 |
|------|---------|---------|------|
| 脏杯子 | 餐桌左侧 | 洗碗机 | 需要清洗的灰色杯子 |
| 餐巾纸 | 餐桌右侧 | 垃圾桶 | 用过的黄色餐巾纸 |

### 1.5 目标容器

| 容器 | 位置 | 接受物品 | 视觉标记 |
|------|------|---------|---------|
| 洗碗机 | 餐厅右侧 (x:3.0) | cup | 蓝色发光标记 |
| 垃圾桶 | 餐厅左侧 (x:-3.0) | tissue | 红色发光标记 |
| 餐桌 | 餐厅中央 | - | 初始物品所在 |

### 1.6 分步引导

| 步骤 | 引导内容 | 类型 |
|------|---------|------|
| 1 | 欢迎提示，介绍桌上物品 | info |
| 2 | 靠近物品按 F 拾取 | info |
| 3 | 按 E 保存记忆，记忆槽有限 | info |
| 4 | 拿着物品靠近目标区按 F 放置 | info |
| 6 | 鼓励提示 | info |
| 8 | 提示洗碗机位置（蓝色标记） | info |
| 10 | 提示垃圾桶位置（红色标记） | info |

### 1.7 记忆测试 (Probes)

| 类型 | 问题 | 选项 |
|------|------|------|
| 位置 | 脏杯子一开始放在哪里？ | 餐桌上 / 洗碗机里 / 垃圾桶里 |
| 位置 | 餐巾纸应该扔到哪里？ | 洗碗机 / 垃圾桶 / 餐桌 |

### 1.8 对话触发

| 触发 | 对话内容 |
|------|---------|
| 关卡开始 | MEM-07 自我介绍，说明记忆故障 |
| 杯子归位 | MEM-07 表扬，提示还剩餐巾纸 |
| 餐巾纸归位 | MEM-07 表扬，预告下一关 |
| 关卡完成 | MEM-07 总结，预告钥匙猫 |

## 关卡 2：出门大作战

### 2.1 任务信息

- **ID**: `task-leave-home`
- **名称**: 出门大作战
- **时间限制**: 10 分钟（600秒）
- **难度**: 中等
- **记忆类型**: 空间记忆、物体位置记忆
- **房间**: 客厅、卧室、玄关

### 2.2 任务背景

早上八点，主人要出门上班。钥匙猫调皮地把钥匙扒拉得到处都是。玩家需要找到钥匙、手机和雨伞，放到玄关托盘上。

### 2.3 目标物品

| 物品 | 初始位置 | 目标位置 | 特殊说明 |
|------|---------|---------|---------|
| 🔑 钥匙 | 客厅茶几 | 玄关托盘 | 钥匙猫会移动它 |
| 📱 手机 | 卧室床头柜抽屉 | 玄关托盘 | 需要打开抽屉 |
| ☂️ 雨伞 | 玄关伞架 | 玄关托盘/伞架 | 已在目标区域附近 |

### 2.4 目标容器

| 容器 | 位置 | 接受物品 |
|------|------|---------|
| 玄关托盘 | 玄关 | 钥匙、手机、雨伞 |
| 茶几 | 客厅 | - |
| 床头柜抽屉 | 卧室 | - |
| 伞架 | 玄关 | 雨伞 |

### 2.5 捣乱事件

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 钥匙猫推钥匙 | 第4步后 | 钥匙移动到客厅角落 |
| 钥匙猫叼钥匙 | 第10步后 | 钥匙移动到卧室 |
| 手机响铃 | 第2步 | 提示手机在卧室抽屉 |
| 主人催促 | 第3步 | 紧迫感提示 |

### 2.6 记忆测试 (Probes)

| 类型 | 问题 | 选项 |
|------|------|------|
| 位置 | 钥匙最初放在哪里？ | 客厅茶几上 / 卧室抽屉 / 厨房台面 / 玄关伞架 |
| 位置 | 手机一开始放在哪里？ | 客厅茶几 / 卧室抽屉 / 厨房台面 / 玄关托盘 |
| 状态 | 手机最初是否可见？ | 可见 / 不可见（藏在抽屉） |
| 计数 | 需要带到玄关的物品一共有几件？ | 2 / 3 / 4 / 5 |

### 2.7 对话触发

| 触发 | 对话内容 |
|------|---------|
| 关卡开始 | 钥匙猫自我介绍，挑战玩家 |
| 进入客厅 | MEM-07 提示茶几上有物品 |
| 进入卧室 | 描述抽屉微微发光 |
| 进入玄关 | MEM-07 提示目标位置 |
| 钥匙归位 | 钥匙猫惊讶，提示手机还在藏着 |
| 钥匙猫事件 | 钥匙猫得意地移动钥匙 |
| 关卡完成 | 钥匙猫认输，预告餐盘精 |

## 关卡 3：餐桌混乱

### 3.1 任务信息

- **ID**: `task-clean-table`（设计阶段）
- **名称**: 餐桌混乱
- **时间限制**: 75秒
- **难度**: 简单
- **记忆类型**: 物体状态记忆、类别规则记忆、空间记忆
- **房间**: 餐厅、厨房、客厅

### 3.2 任务背景

早餐后留下一桌子狼藉，调皮的餐盘精总爱把脏盘子偷偷放回桌上！玩家需要快速识别并归类物品。

### 3.3 目标物品

| 物品 | 类别 | 状态 | 目标位置 |
|------|------|------|---------|
| 脏杯子 | cup | dirty | 洗碗机/水槽 |
| 干净杯子 | cup | clean | 不扔进垃圾桶即可 |
| 脏盘子 | plate | dirty | 洗碗机/水槽 |
| 餐巾纸 | tissue | - | 垃圾桶 |
| 遥控器 | remote | - | 客厅茶几 |

### 3.4 目标容器

| 容器 | 位置 | 接受物品 |
|------|------|---------|
| 洗碗机 | 厨房 | cup, plate |
| 水槽 | 厨房 | cup, plate |
| 垃圾桶 | 厨房 | tissue |
| 客厅茶几 | 客厅 | remote |

### 3.5 捣乱事件

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 餐盘精放回盘子 | 第8步 | 脏盘子回到餐桌 |
| 餐盘精放回杯子 | 第12步 | 脏杯子回到餐桌 |
| 相似杯子提示 | 第3步 | 系统提醒区分颜色 |
| 室友消息 | 第4步 | 提醒注意区分杯子 |

### 3.6 记忆测试 (Probes)

| 类型 | 问题 | 选项 |
|------|------|------|
| 计数 | 餐桌上一共有几个杯子？ | 1 / 2 / 3 / 4 |
| 物体识别 | 哪个杯子需要清洗？ | 白色亮杯 / 灰色杯子 |
| 位置 | 餐巾纸应该扔到哪个容器？ | 洗碗机 / 水槽 / 垃圾桶 |
| 位置 | 遥控器应该放到哪里？ | 洗碗机 / 水槽 / 垃圾桶 / 客厅茶几 |

## 关卡 4：洗衣幽灵

### 4.1 任务信息

- **ID**: `task-laundry-sort`
- **名称**: 洗衣幽灵
- **难度**: 困难
- **记忆类型**: 配对记忆、程序记忆
- **房间**: 洗衣房

### 4.2 任务背景

洗衣房里传来窸窸窣窣的声音...三只篮子整齐地排成一排，空气中似乎漂浮着什么。袜子幽灵会交换篮子里的袜子，考验玩家的配对记忆能力。

### 4.3 核心玩法

- 识别成对的袜子（颜色、图案匹配）
- 将配对的袜子放入正确的抽屉
- 袜子幽灵会交换袜子位置
- 需要记住袜子的配对关系

### 4.4 目标物品

| 物品 | 类别 | 初始位置 | 目标位置 |
|------|------|---------|---------|
| 白袜子（左） | sock | 洗衣篮 | 白色衣物篮 |
| 白袜子（右） | sock | 洗衣篮 | 白色衣物篮 |
| 深色袜子（左） | sock | 洗衣篮 | 深色衣物篮 |
| 深色袜子（右） | sock | 洗衣篮 | 深色衣物篮 |
| 毛巾 | towel | 洗衣篮 | 毛巾篮 |
| 彩色衬衫 | shirt | 洗衣篮 | 白色衣物篮（浅色） |

### 4.5 目标容器

| 容器 | 位置 | 接受物品 |
|------|------|---------|
| 白色衣物篮 | 洗衣房左侧 | 白色袜子、彩色衬衫 |
| 深色衣物篮 | 洗衣房中间 | 深色袜子 |
| 毛巾篮 | 洗衣房右侧 | 毛巾 |

### 4.6 捣乱事件

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 袜子幽灵移动衣物 | 第5步 | 白袜子被移动到毛巾篮附近 |
| 袜子幽灵交换篮子位置 | 第8步 | 白色和深色篮子位置互换 |

### 4.7 对话触发

| 触发 | 对话内容 |
|------|---------|
| 关卡开始 | 袜子幽灵自我介绍，说明分类规则 |
| 幽灵事件 | 袜子幽灵得意地移动袜子 |
| 关卡完成 | 袜子幽灵认输，预告时间循环 |

## 关卡 5：早餐时间循环

### 5.1 任务信息

- **ID**: `task-breakfast`
- **名称**: 早餐时间循环
- **难度**: 困难
- **记忆类型**: 时间记忆、空间记忆、程序记忆
- **房间**: 厨房

### 5.2 任务背景

深夜十一点，厨房的闹钟突然响了...时间好像在倒转。玩家需要在时间循环中按照正确流程准备早餐并归位物品。

### 5.3 核心玩法

- **第一阶段**：按顺序取出牛奶→麦片→碗→杯子放到餐桌
- **第二阶段**：全部归位并关好容器（牛奶回冰箱、杯碗放洗碗机、麦片放橱柜）
- 早餐闹钟会把麦片移到上层橱柜
- 必须严格按照流程图操作

### 5.4 目标物品

| 物品 | 初始位置 | 第一阶段目标 | 第二阶段目标 |
|------|---------|-------------|-------------|
| 牛奶 | 冰箱 | 餐桌 | 冰箱（关好门） |
| 麦片 | 下层橱柜 | 餐桌 | 上层橱柜（关好门） |
| 碗 | 洗碗机 | 餐桌 | 洗碗机（关好门） |
| 杯子 | 橱柜 | 餐桌 | 洗碗机（关好门） |

### 5.5 目标容器

| 容器 | 位置 | 接受物品 |
|------|------|---------|
| 冰箱 | 厨房左侧 | 牛奶 |
| 洗碗机 | 厨房右侧 | 碗、杯子 |
| 餐桌 | 厨房中央 | 所有物品（第一阶段） |
| 下层橱柜 | 厨房右侧下方 | 麦片（初始位置） |
| 上层橱柜 | 厨房右侧上方 | 麦片（归位位置） |

### 5.6 捣乱事件

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 早餐闹钟移动麦片 | 第7步 | 麦片被移到上层橱柜 |
| 时间循环警告 | 第3步 | 提示流程错误需重新开始 |

### 5.7 对话触发

| 触发 | 对话内容 |
|------|---------|
| 关卡开始 | 早餐闹钟发出错误提示，说明时间循环 |
| 准备阶段完成 | 早餐闹钟提示进入归位阶段 |
| 麦片移动事件 | 早餐闹钟提示麦片放错地方 |
| 关卡完成 | 时间循环打破，MEM-07 总结全篇故事 |

## 关卡设计原则

### 难度递进

```
关卡1（教学）→ 基础操作学习，无捣乱事件
关卡2（出门）→ 空间记忆，单一捣乱者（钥匙猫）
关卡3（餐桌）→ 状态识别，多物品处理，餐盘精捣乱
关卡4（洗衣）→ 配对记忆，复杂规则，袜子幽灵捣乱
关卡5（早餐）→ 时间循环，高阶策略，流程记忆
```

### 记忆类型覆盖

每个关卡侧重不同的记忆类型，全面训练玩家的记忆能力：

| 记忆类型 | 训练关卡 | 核心挑战 |
|---------|---------|---------|
| 空间记忆 | 关卡1、2、5 | 记住物品位置 |
| 物体记忆 | 关卡1、3 | 识别物品属性和状态 |
| 配对记忆 | 关卡4 | 记住物品配对关系 |
| 程序记忆 | 关卡3、4、5 | 记住操作序列和规则 |
| 时间记忆 | 关卡5 | 记住时间变化和循环 |

### 叙事连贯性

四个关卡串联成完整的一天故事线，每个关卡结束都有 NPC 预告下一关内容：

1. **初次整理** → MEM-07 预告钥匙猫
2. **出门大作战** → 钥匙猫预告餐盘精
3. **洗衣幽灵** → 袜子幽灵预告时间循环
4. **早餐时间循环** → MEM-07 总结全篇

### 房间物品合理性

每个房间的物品都符合现实生活逻辑：

| 房间 | 合理物品 |
|------|---------|
| 餐厅 | 桌子、椅子、餐具、垃圾桶、洗碗机 |
| 客厅 | 茶几、沙发、电视、遥控器 |
| 卧室 | 床、床头柜、抽屉、手机 |
| 玄关 | 托盘、伞架、钥匙 |
| 洗衣房 | 洗衣机、洗衣篮、袜子、毛巾 |
| 厨房 | 冰箱、橱柜、餐桌、牛奶、麦片、碗、杯子 |
