# HomeMem Arena - 游戏设计概览

## 1. 游戏简介

HomeMem Arena 是一款结合记忆训练与家务模拟的 3D 网页游戏。玩家扮演记忆有限的家政机器人「小橡」，在会捣乱的房子里完成家务任务，应对调皮的记忆小妖。

**核心定位**：
- 🎮 一款让你顺便练记忆的 3D 网页小游戏
- 🔬 同时也是一个轻量研究平台，记录完整的观察、操作、记忆更新与环境扰动数据

## 2. 核心玩法循环

```
观察 → 决策 → 行动 → 反馈 → 记忆管理
```

1. **观察**：探索房间，发现散落的物品
2. **决策**：决定拾取什么、保存什么记忆
3. **行动**：拾取物品、放入容器、打开抽屉
4. **反馈**：即时音效、视觉反馈、得分变化
5. **记忆管理**：策略性保存/更新/删除记忆

## 3. 游戏特色

### 3.1 有限记忆槽
- 只能同时记住 3 件物品
- 需要策略性地选择保存什么
- 记忆会随时间过期

### 3.2 捣乱事件
- **钥匙猫**：会移动钥匙位置
- **餐盘精**：把脏盘子放回桌子
- **袜子幽灵**：交换篮子里的袜子
- **时间循环**：让物品回到初始位置

### 3.3 混乱值系统
- 混乱值越高，环境越不稳定
- 物品更易被移动，记忆更易过期
- 考验临场应变与记忆策略

### 3.4 多维度评分
- 速度、连击、记忆测试正确率
- 多种策略路径拿高分

### 3.5 一天时间线
- 清晨 → 上午 → 下午 → 深夜
- 四个关卡串联完整叙事

## 4. 目标受众

| 人群 | 核心需求 |
|------|---------|
| 休闲玩家 | 有趣的记忆挑战游戏 |
| 学生/脑力训练爱好者 | 科学的记忆训练工具 |
| 研究者 | 可复现的记忆策略研究平台 |

## 5. 设计目标

HomeMem Arena 的功能只有同时满足以下三点才算完成：

1. **工程正确** — 关卡可完成，状态逻辑自洽，不同输入方式产生一致结果
2. **游戏好玩** — 目标明确、反馈及时、挑战来自有意义的记忆取舍，熟练后可以优化策略
3. **研究有效** — 环境扰动可控、Session 数据完整、实验可复现、指标能回答长期记忆问题
