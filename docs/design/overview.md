# 记忆宅邸：失忆管家 - 游戏设计概览

## 1. 游戏简介

**记忆宅邸：失忆管家**是一款结合记忆训练与家务模拟的 3D 网页游戏。玩家扮演记忆模块出了故障的家务机器人 **MEM-07**，只能同时记住 3 件物品的位置。在一栋「会捣乱」的房子里，玩家需要限时完成各种家务任务，应对调皮的 NPC（钥匙猫、餐盘精、袜子幽灵、早餐闹钟），在轻松闯关的过程中锻炼空间记忆、物体状态记忆和流程记忆。

**核心定位**：
- 🎮 一款让你在玩游戏时顺便练记忆的 3D 网页小游戏
- 🔬 同时也是一个轻量研究平台，记录完整的观察、操作、记忆更新与环境扰动数据

**灵感来源**：家务机器人执行长程任务时遇到的记忆问题——物品会被移动，房间状态会变化，任务步骤也可能被打断。我们把这个方向转化成更轻松的游戏设定，让玩家亲身体验「记忆有限」是什么感觉。

## 2. 核心玩法循环

```
观察 → 决策 → 行动 → 反馈 → 记忆管理
```

1. **观察**：探索房间，发现散落的物品
2. **决策**：决定拾取什么、保存什么记忆
3. **行动**：拾取物品、放入容器、打开抽屉
4. **反馈**：即时音效、视觉反馈、得分变化、NPC对话
5. **记忆管理**：策略性保存/更新/删除记忆

## 3. 游戏特色

### 3.1 有限记忆槽
- 只能同时记住 3 件物品
- 需要策略性地选择保存什么
- 记忆会随时间过期

### 3.2 捣乱事件与 NPC
- **钥匙猫**：会把钥匙扒拉到各处
- **餐盘精**：把脏盘子偷偷放回桌上
- **袜子幽灵**：交换篮子里的袜子、藏起袜子
- **早餐闹钟**：时间循环，物品回到初始位置

### 3.3 混乱值系统
- 混乱值越高，环境越不稳定
- 物品更易被移动，记忆更易过期
- 考验临场应变与记忆策略

### 3.4 多维度评分
- 速度、连击、记忆测试正确率
- 多种策略路径拿高分
- S/A/B/C/D 星级评价

### 3.5 完整叙事线
- 四个关卡串联成一天的故事线：清晨 → 上午 → 下午 → 深夜
- 每关结束预告下一关内容，形成连贯叙事
- NPC 对话增强故事代入感

## 4. 目标受众

| 人群 | 核心需求 |
|------|---------|
| 休闲玩家 | 有趣的记忆挑战游戏，单局 3-5 分钟，适合碎片时间 |
| AI 与科技爱好者 | 通过游戏直观理解「为什么机器人需要记忆」 |
| 记忆训练爱好者 | 用游戏化方式自然练习各种记忆类型，比传统训练更有趣 |
| 研究者 | 可复现的记忆策略研究平台 |

## 5. 设计目标

游戏的功能只有同时满足以下三点才算完成：

1. **工程正确** — 关卡可完成，状态逻辑自洽，不同输入方式产生一致结果
2. **游戏好玩** — 目标明确、反馈及时、挑战来自有意义的记忆取舍，熟练后可以优化策略
3. **研究有效** — 环境扰动可控、Session 数据完整、实验可复现、指标能回答长期记忆问题
