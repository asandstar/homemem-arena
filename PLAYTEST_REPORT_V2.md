## 1. 试玩路径

### 第一关「出门大作战」完整流程
1. 进入游戏 → 点击"开始任务"初始化音频
2. 在客厅找到钥匙（茶几上，黄色）
3. 走向卧室 → 找到床头柜抽屉 → 按 F 打开 → 取出手机
4. 走向玄关 → 找到雨伞（伞架旁，红色）
5. 按 E 保存记忆（测试 memory_save 音效）
6. 将物品放到玄关托盘
7. 等待猫事件和手机响铃事件触发
8. 完成所有目标 → 进入结算页

### 第二关「餐桌混乱」检查
1. 从餐厅开始
2. 检查餐桌上的杯子、盘子、遥控器模型颜色
3. 走向厨房 → 检查洗碗机、水槽位置
4. 走向客厅 → 检查茶几位置

### 第三关「洗衣幽灵」检查
1. 从洗衣房开始（位置 x:18）
2. 检查衣物、篮子位置

### 第四关「早餐时间循环」检查
1. 从厨房开始
2. 检查冰箱、橱柜、餐桌位置

## 4. 潜在问题（代码分析）

### ⚠️ 第二关容器位置问题
**文件**: `src/data/tasks/clean-table.ts`

```typescript
// 厨房 - 洗碗机位置（世界坐标计算）
position: { x: 5.0, y: 0.4, z: -2.0 }  // 相对于厨房中心
```

厨房中心在 `{ x: 6, y: 0, z: 0 }`，所以洗碗机世界坐标约为 `{ x: 11, z: -2 }`。

**问题**: 容器位置使用的是相对坐标，但代码中 `inferContainerRoom` 函数推断房间时可能不准确。需要验证：
- 玩家在厨房能否看到洗碗机
- 距离判定是否正确（2.5m 范围）

### ⚠️ 第三关洗衣房初始位置
**文件**: `src/data/tasks/laundry-sort.ts`

```typescript
rooms: ['laundry'],
objects: [
  { initialPosition: { x: -3.0, y: 0.4, z: 1.0 } }  // 白衬衫
  { initialPosition: { x: 0, y: 0.4, z: 1.0 } }      // 黑 T 恤
  { initialPosition: { x: 3.0, y: 0.4, z: 1.0 } }    // 大浴巾
]
```

洗衣房现在中心在 `{ x: 18, z: 0 }`，尺寸 `{ x: 6, z: 6 }`。物体的相对位置需要在房间范围内：

- 白衬衫相对位置 x: -3 → 世界坐标 x: 15 ✅（在房间内）
- 黑 T 恤相对位置 x: 0 → 世界坐标 x: 18 ✅（房间中心）
- 大浴巾相对位置 x: 3 → 世界坐标 x: 21 ✅（在房间内）

**问题**: 篮子位置：
```typescript
position: { x: -3.0, y: 0.25, z: -2.0 }  // 白色篮子
position: { x: 0, y: 0.25, z: -2.0 }      // 深色篮子
position: { x: 3.0, y: 0.25, z: -2.0 }    // 毛巾篮
```

世界坐标：
- 白色篮子 x: 15, z: -2 ✅
- 深色篮子 x: 18, z: -2 ✅
- 毛巾篮 x: 21, z: -2 ✅

看起来都在房间范围内，但需要验证玩家能否看到和交互。

### ⚠️ 混乱值警告音效限流
**文件**: `src/audio/sfx.ts`

```typescript
let lastChaosWarningTime = 0
export function playChaosWarning(): void {
  const now = Date.now()
  if (now - lastChaosWarningTime < 3000) return  // 3秒限流
  lastChaosWarningTime = now
  playSfx('chaos_warning')
}
```

**问题**: 如果混乱值快速从 79 跳到 82（例如事件触发），可能只触发一次警告。这是预期行为，但需要验证。
