## 不做的事

1. **GitHub Pages 可能还没部署完成**：Actions 刚 push 可能还在运行中，但地址是确定的，可以先写上。
2. **保持双定位平衡**：项目既是游戏也是研究平台，README 开头需要同时满足两种访客的需求，不能完全倒向一边。
3. **链接正确性**：GitHub Pages 的 base path 是 `/homemem-arena/`，确保链接写对。
