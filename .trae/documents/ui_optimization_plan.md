# HomeMem Arena UI 优化计划

## 问题分析

### 1. WASD 移动逻辑错误
- 当前只处理前后移动（Z轴），缺少左右移动（X轴）
- 移动方向可能与视角方向不一致

### 2. 文字清晰度问题
- 3D场景中的物体标签和房间名称颜色不够清晰
- 黄色文字在某些背景下难以辨认

### 3. 快捷键提示不清晰
- ESC键退出功能未提示
- 快捷键排版杂乱，分组不清晰

### 4. UI布局不够游戏化
- 小地图在右下角，不符合游戏习惯
- 缺少HUD边框和视觉层次

### 5. RoboMME科研设计对齐
- 需要更清晰的任务状态指示
- 需要更好的观察和交互反馈

## 优化方案

### 一、修复 WASD 移动逻辑
1. 添加左右移动支持（X轴）
2. 确保移动方向与相机视角一致
3. 修复方向反转问题

### 二、改进文字清晰度
1. 调整3D场景中文字颜色为白色或高对比度颜色
2. 添加文字背景阴影
3. 增加文字大小

### 三、优化快捷键提示
1. 添加ESC键说明（退出弹窗/释放鼠标）
2. 分组显示快捷键（移动、视图、面板、交互）
3. 更清晰的排版

### 四、游戏化UI布局
1. 将小地图移到左上角（任务目标下方）
2. 添加HUD边框和视觉效果
3. 改进状态栏设计

### 五、增强交互反馈
1. 添加鼠标锁定状态提示
2. 添加移动方向指示器
3. 改进任务进度可视化

## 涉及文件

### 核心修改
- `src/components/arena3d/FirstPersonControls.tsx` - 修复移动逻辑
- `src/components/arena3d/HUD.tsx` - 优化UI布局和快捷键提示
- `src/components/arena3d/Object3D.tsx` - 改进文字标签清晰度
- `src/components/arena3d/Room3D.tsx` - 改进房间名称显示
- `src/pages/ArenaPage.tsx` - 添加ESC键关闭弹窗

### 辅助修改
- `src/store/useGameStore.ts` - 添加左右移动方法（如需要）

## 实施步骤

### Step 1: 修复移动逻辑
- 检查useGameStore中的moveForward方法
- 添加moveSide方法或修改moveForward支持X轴移动
- 确保移动方向与相机视角一致

### Step 2: 添加ESC键功能
- 在ArenaPage中添加ESC键关闭任务简报
- 在HUD中添加ESC键释放鼠标锁定

### Step 3: 优化快捷键提示
- 重新组织快捷键分组
- 添加图标和更好的视觉效果

### Step 4: 移动小地图位置
- 将小地图从右下角移到左上角
- 调整控制面板位置

### Step 5: 改进文字清晰度
- 修改Object3D和Room3D中的文字颜色和样式

## 风险处理

1. **移动逻辑修改可能影响现有功能**：测试所有任务的移动功能
2. **UI布局调整可能导致遮挡**：仔细检查各组件位置关系
3. **文字渲染性能**：保持文字数量合理，避免过多标签

## RoboMME设计参考

根据RoboMME的设计理念，我们将：
- 提供清晰的任务进度可视化
- 增强观察和记忆的反馈机制
- 确保交互操作有明确的视觉反馈
- 保持简洁而信息丰富的HUD设计
