# HomeMem Arena v2 优化计划

## 问题分析

### 1. 组件遮挡问题
- 底部事件日志、操作提示和右下角控制面板互相遮挡
- 小地图被控制面板遮挡
- 顶部进度提示与 HUD 重叠

### 2. 3D 场景简陋
- 物体仍然显示为方块（自定义几何未生效）
- 缺乏房间边界和门的可视化
- 场景光照和材质过于简单

### 3. 浏览器错误
- `net::ERR_ABORTED` 错误表明某些模块加载失败（可能是热更新问题）

### 4. UI/UX 问题
- 任务目标面板不够清晰
- 缺少房间名称显示
- 物体交互提示不明显

## 优化方案

### 一、UI 布局重构
1. **重新组织 HUD 布局**，消除组件遮挡
2. **将控制面板移到顶部或左侧**，避免与小地图冲突
3. **事件日志缩小并调整位置**
4. **操作提示改为悬浮卡片**

### 二、3D 场景增强
1. **确保自定义几何生效**（牛奶盒、麦片盒、杯子、碗）
2. **添加房间边界和门的可视化**
3. **增强场景光照**
4. **添加房间地面纹理区分**

### 三、交互体验提升
1. **添加物体名称标签**（hover 时显示）
2. **添加房间名称显示在场景中**
3. **增强选中/高亮效果**
4. **添加视角平滑切换动画**

### 四、任务目标面板优化
1. **更清晰的目标列表**
2. **进度条可视化**
3. **当前目标高亮**

## 涉及文件

### UI 布局
- `src/components/arena3d/HUD.tsx` - HUD 布局重构
- `src/pages/ArenaPage.tsx` - 控制面板位置调整

### 3D 场景
- `src/components/arena3d/Scene3D.tsx` - 场景光照和房间可视化
- `src/components/arena3d/Object3D.tsx` - 自定义几何渲染
- `src/components/arena3d/ObjectGeometries.tsx` - 几何定义
- `src/components/arena3d/Room3D.tsx` - 房间边界和门

### 交互体验
- `src/components/arena3d/Scene3D.tsx` - 添加物体标签和高亮
- `src/store/useGameStore.ts` - 视角切换动画

- 重新组织组件位置

### Step 3: 增强 3D 场景
- 添加房间边界

### Step 4: 优化交互体验

## 风险处理

1. **热更新失败**：重新启动开发服务器
2. **几何渲染问题**：回退到基础几何并逐步调试
3. **性能问题**：保持低多边形风格，避免复杂几何体
