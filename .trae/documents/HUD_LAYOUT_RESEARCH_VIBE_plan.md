# HUD 布局优化 + 研究意味增强计划

## 核心目标

1. **消除模块重叠**：重新设计 HUD 布局，确保所有模块不重叠
2. **符合玩家习惯**：遵循主流游戏的 UI 布局规范
3. **增强研究意味**：在不改变玩法的前提下，添加研究风格的视觉元素

## 新布局设计（遵循游戏 UI 最佳实践）

```
┌─────────────────────────────────────────────────────────────┐
│  [任务面板]                              [小地图] [音效]    │
│  左上角                                   右上角            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    [得分] [评级] [时间]                      │
│                   [混乱值] [进度条]                          │
│                   顶部中央（核心状态）                        │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  [事件日志]                              [操作提示]          │
│  左下角                                   右下角            │
├─────────────────────────────────────────────────────────────┤
│                    [记忆槽]                                 │
│                   底部中央（核心交互）                        │
└─────────────────────────────────────────────────────────────┘
```

## 布局调整细节

### 顶部区域
- **左上角**：任务面板（300px 宽度，可折叠）
- **右上角**：小地图（200px 宽度）+ 音效开关 + 重新开始按钮
- **顶部中央**：得分、评级、时间、混乱值、进度条（合并为一个紧凑面板）

### 底部区域
- **左下角**：事件日志（可折叠）
- **右下角**：操作提示（仅在 playing 阶段显示）
- **底部中央**：记忆槽（核心交互，始终可见）

### 浮动元素
- **顶部中央下方**：Toast 通知（不遮挡核心信息）
- **屏幕中央**：Combo 动画、成功/失败闪烁

## HUD 布局优化

### 1. 消除重叠
- 每个区域有明确的边界，不重叠
- 使用 z-index 确保层级清晰
- 紧凑模式下自动调整大小和位置

### 2. 符合玩家习惯
- 得分在顶部中央，符合大多数游戏的设计
- 小地图在右上角，便于快速查看
- 任务面板在左上角，便于随时查看目标
- 记忆槽在底部中央，便于快速操作

## 研究意味增强

### 1. 记忆类型可视化
- 在记忆槽中添加记忆类型图标（空间/物体/时间/程序）
- 颜色编码：空间记忆-绿色、物体记忆-紫色、时间记忆-蓝色、程序记忆-橙色
- 不增加操作复杂度，只是视觉增强

### 2. 研究风格包装
- HUD 使用科技感配色（深色背景 + 霓虹色调）
- 记忆槽添加「记忆置信度」进度条（已实现）
- 结果页面添加「记忆类型分析报告」

## 关键设计决策

1. **布局优先级**：核心状态（得分、混乱值）> 核心交互（记忆槽）> 辅助信息（事件日志、操作提示）
2. **记忆类型颜色编码**：
   - 空间记忆：绿色
   - 物体记忆：紫色
   - 时间记忆：蓝色
   - 程序记忆：橙色
3. **紧凑模式**：自动缩小面板尺寸，调整位置避免重叠
4. **可折叠性**：任务面板、事件日志、小地图都支持折叠，减少屏幕占用

## 保持游戏性的关键

1. **不增加操作复杂度**：记忆类型可视化只是视觉增强，不改变操作方式
2. **不改变核心玩法**：拾取/放置/记忆系统保持不变
3. **不增加学习成本**：布局符合玩家习惯，上手容易
4. **不打断游戏节奏**：所有面板都支持折叠，玩家可以专注游戏
