# 剧情系统重构计划（奇幻轻松风格）

## 一、核心问题诊断

用户反馈：
- ✅ 喜欢之前轻松有趣的风格（猫咪、幽灵、时间循环都挺有意思）
- ❌ 问题在于：AI 味道重、剧情零散、不知道"为什么要做这些事"
- ❌ 缺乏统一世界观和角色动机，代入感弱

**解决方案**：保留轻松有趣的调性，给它套一个统一的奇幻世界观骨架，让四个关卡串联成一个完整故事。

---

## 二、世界观设定（奇幻轻松向）

### 核心设定：「记忆之家」与家政小精灵

**世界背景**：
- 在人类世界的缝隙里，存在着一种靠"记忆"为生的小精灵
- 每一个充满回忆的老房子里，都住着一只家政小精灵
- 小精灵靠整理房间、维护物品秩序来"喂养"房子里的记忆
- 如果房子太久没人整理，记忆就会变得混乱，东西会自己乱跑

**玩家角色**：
- 你是一只新来的实习家政小精灵，名字叫 **「小橡」**（或者玩家自己取）
- 你的第一份工作：照看一栋叫做「回声屋」的老房子
- 房子里有四只调皮的记忆小妖，它们总爱把东西藏起来、移来移去
- 你的魔力有限——一次只能记住 3 样东西的位置（对应记忆槽系统）

**混乱值的真正含义**：
- 不是"房间乱"，而是**记忆扰动值**
- 记忆小妖越活跃，扰动值越高
- 扰动太高的话，房子的空间会开始扭曲
- 严重时连你自己都会忘记东西放在哪（游戏失败）

### 四只记忆小妖（四个关卡的反派/主角）

| 小妖 | 性格 | 对应关卡 | 技能 |
|------|------|----------|------|
| **钥匙猫** | 好奇贪玩，喜欢把小东西扒拉到地上 | 出门大作战 | 把钥匙推到地上、藏起来 |
| **餐盘精** | 恶作剧大王，总觉得干净盘子太无聊 | 餐桌混乱 | 把脏盘子放回桌上、混淆干净和脏的 |
| **袜子幽灵** | 害羞内向，喜欢躲在衣物里 | 洗衣幽灵 | 藏袜子、交换篮子位置、变出神秘衣物 |
| **早餐闹钟** | 强迫症，每天早上必须重复同样的事 | 早餐时间循环 | 时间循环、移动物品、自动关冰箱门 |

### 整体故事线
四个关卡不是孤立的，而是一天之内发生的事：
1. **早晨** - 主人要出门，钥匙猫开始捣乱（出门大作战）
2. **上午** - 室友吃完早餐跑了，餐盘精开始搞事（餐桌混乱）
3. **下午** - 洗衣房里，袜子幽灵在玩捉迷藏（洗衣幽灵）
4. **晚上** - 早餐闹钟卡在了时间循环里（早餐时间循环）

每通过一关，你和这些小妖的关系就更近一步——它们不是反派，只是太寂寞了，想找人玩。

## 三、剧情系统设计

### 1. 剧情层次

**第一层：世界观铺垫**
- 首页：介绍"家政小精灵"的设定
- 任务选择：四个任务按时间线排列，有剧情顺序
- 每关开始前：一段开场对话/独白

**第二层：关卡内叙事**
- 开场：小妖登场，介绍本关玩法
- 事件：每次触发事件都有小妖的"台词"
- 中期：混乱值升高时，小妖变本加厉
- 完成：小妖认输/和你成为朋友
- 失败：小妖得意地嘲笑你

**第三层：整体串联**
- 每关结束后解锁一小段过场
- 通关全部四关后有一个大结局
- 隐藏：所有关卡 S 级通关有彩蛋

### 2. 对话系统功能
- 对话框组件：带角色头像、名字、台词
- 打字机效果：逐字显示
- 选项系统：少量对话选择（不影响主线，只是增加趣味）
- 跳过功能：可以快速跳过剧情

### 3. 文案风格
- 保持轻松有趣、带点小幽默
- 角色有自己的语气和性格
- 有"主线剧情"但不冗长
- 不破坏游戏节奏

## 四、实施步骤

### 阶段一：世界观 + 文案重写（第一优先级）
**目标**：不改动代码逻辑，只改文案，立刻提升代入感

1. **首页重写**：介绍"家政小精灵"设定，替代现在的干巴巴介绍
2. **任务选择页重写**：四个任务按时间线排列，每个有剧情简介和小妖头像
3. **四个关卡文案重写**：
   - briefing 改写成"任务委托"风格
   - systemPrompt 改成小精灵内心独白
   - scripted events 的 message 改成小妖台词
   - completionText / failureText 改成有角色感的文字
4. **HUD 文案微调**：把"混乱值"、"记忆槽"等术语用更奇幻的方式表达（但保留功能）

### 阶段二：对话系统基础
1. 创建 `DialogueBox` 组件（对话框 + 打字机效果）
2. 创建对话数据格式（说话人、文本、表情）
3. 支持剧情触发（开场、关键事件、结局）

### 阶段三：剧情事件嵌入
1. 每关开场加一段小妖登场的对话
2. 第一次触发事件时加对话
3. 高混乱值时加小妖的嘲讽/挑衅
4. 完成/失败时加角色互动对话

### 阶段四：整体叙事串联
1. 关卡之间加简短过场
2. 通关全部四关后有大结局
3. 根据最终评级有不同结局
4. 隐藏彩蛋

## 六、文案风格示例

### 任务简报（Before → After）

Before:
```
📝 早上好 MEM-07！我快迟到了！！
钥匙在客厅茶几上，手机在卧室，雨伞在厨房。
帮我把它们放到玄关托盘上，我五分钟后出门。
—— 主人
P.S. 别理那只猫。
```

After:
```
🌅 清晨 · 回声屋

"喂——新来的小精灵！"
一只橘白相间的猫咪从窗帘后探出头。

"我是这房子的钥匙守护者，
想把东西放对地方？
先找得到再说喵~"

⚠️ 钥匙猫已经开始行动了
⏰ 限时 3 分钟
🎯 目标：钥匙、手机、雨伞 → 玄关托盘
```

### 事件提示（Before → After）

Before:
```
🐱 幽灵猫咪！它把白袜子扒拉到了毛巾篮附近，快找回来！
```

After:
```
👻 袜子幽灵的声音：
"嘿嘿，找不到了吧~
这些白乎乎的东西，
本来就是用来捉迷藏的呀~"
```

## 七、风险与注意事项

1. **不影响核心玩法**：剧情是锦上添花，不改变游戏机制
2. **可跳过**：对话都能跳过，不影响想快速玩的玩家
3. **风格统一**：四个关卡用同一套角色和世界观
4. **逐步迭代**：先改文案看效果，满意了再做对话系统
5. **工作量可控**：阶段一只是改文案，风险低

---

## 八、优先级

**✅ P0 先做**：阶段一（文案重写 + 世界观）
- 工作量小，见效快
- 立刻提升游戏质感
- 不改动核心代码

**📅 P1 后续**：阶段二 + 三（对话系统）
- 有了文案基础后再加交互
- 打字机效果、角色对话

**🌟 P2 远期**：阶段四（整体叙事 + 多结局）
- 四个关卡串联成完整故事
- 需要更多内容创作
