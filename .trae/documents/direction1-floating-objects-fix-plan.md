# 方向1：物体与家具摆放悬空问题修复计划

## 问题概述

经过代码审查，发现游戏中多个家具和装饰物存在 Y 轴位置不正确的问题，导致物体看起来悬浮在空中或嵌入地面/台面。核心问题集中在：

1. **餐厅场景**：餐桌上的杯子和盘子明显悬浮
2. **卧室场景**：台灯放在地上而非床头柜/书桌上
3. **厨房场景**：洗洁精瓶子部分嵌入台面
4. **多个场景**：植物底部未贴地

## 根因分析

1. **硬编码位置错误**：Room3D.tsx 中装饰物体的 Y 坐标是手动硬编码的，没有按照「底部对齐表面」的原则统一计算
2. **缺少统一锚定逻辑**：装饰性物体（非任务关键物体）直接在 Room3D.tsx 中定义，没有走 `placement.ts` 的表面锚定系统
3. **Three.js 坐标系混淆**：Three.js 几何体默认中心在原点，position.y 是中心点高度，容易与「底部高度」混淆

## 修复方案

### 原则

1. **底部对齐原则**：所有站立在地面/台面上的物体，其底部必须与表面齐平
2. **统一计算公式**：`position.y = surfaceHeight + size.y / 2`
3. **表面高度表**：
   - 地面：0
   - 餐桌桌面：0.9m（餐桌高 0.9m）
   - 床头柜台面：0.55m（床头柜高 0.55m）
   - 书桌桌面：0.75m（书桌高 0.75m）
   - 厨房台面：0.9m（橱柜高 0.9m）
   - 洗衣机顶部：1.1m（洗衣机高 1.1m）

### 具体修改

#### 1. 餐厅场景修复（`renderDining`）

- 4 个杯子：`position.y` 从 1.28 → 0.95
- 盘子：`position.y` 从 1.25 → 0.91
- 角落植物：`position.y` 从 0.5 → 0.35

#### 2. 卧室场景修复（`renderBedroom`）

- 左床头柜台灯：`position.y` 从 0.225 → 0.775
- 右床头柜台灯：`position.y` 从 0.225 → 0.775
- 书桌台灯：`position.y` 从 0.2 → 0.95

#### 3. 厨房场景修复（`renderKitchen`）

- 洗洁精瓶子：`position.y` 从 0.9 → 1.0

#### 4. 洗衣房场景修复（`renderLaundry`）

- 洗衣液瓶子：`position.y` 从 0.9 → 1.225

### 可选优化

考虑到未来可能继续添加装饰物体，建议：
- 在 Room3D 中添加一个工具函数 `placeOnSurface(surfaceY, sizeY)` 来计算正确的 Y 坐标
- 将装饰物体数据化为配置，而非硬编码 JSX

本次修复暂不实施，保持改动最小化。

## 风险与注意事项

1. **模型原点差异**：部分 GLB 模型的原点可能不在底部中心，需要确认 fallback 几何体的原点位置
   - 当前所有 fallback 几何体都是中心在原点的 box/cylinder，position.y = size.y/2 正确

2. **视觉微调**：某些物体（如台灯）可能需要根据实际视觉效果微调高度，上述数值是理论计算值

3. **范围限定**：本次只修复 Room3D.tsx 中的装饰物体，任务关键物体（通过 Container3D/Object3D 渲染的）已通过 placement.ts 的表面锚定系统处理，理论上没问题，但仍需在验证步骤中确认
