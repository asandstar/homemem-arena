# Foundation Pass - Verification Checklist

## 基础操作

- [ ] W 键向屏幕前方移动
- [ ] S 键向后退
- [ ] A 键向左平移
- [ ] D 键向右平移
- [ ] 鼠标左右拖动可以左右转向
- [ ] 鼠标上下拖动可以抬头低头
- [ ] pitch（上下视角）限制在合理范围内（不会翻跟头）
- [ ] 画面不抖（无 camera 和 rotation 互相追逐）
- [ ] 连续移动 30 秒无明显卡顿
- [ ] V 键切换视角后，WASD 方向仍然正确

## 碰撞与穿墙

- [ ] 玩家不能穿过外墙
- [ ] 撞墙时可以沿墙滑动，不会直接卡死
- [ ] 可以自然通过门洞进入相邻房间
- [ ] 门口不会抖动或反复切换房间
- [ ] 第一关能在客厅、卧室、玄关之间稳定移动
- [ ] 第二关能在餐厅、厨房之间稳定移动
- [ ] 不会被家具推出墙外

## 物体摆放

- [ ] 第二关餐桌上的杯子不悬空
- [ ] 第二关餐桌上的盘子不悬空
- [ ] 第二关餐桌上的遥控器不悬空
- [ ] 第二关餐桌上的垃圾不悬空
- [ ] 第一关钥匙不悬空
- [ ] 第一关手机不悬空
- [ ] 第一关雨伞不悬空
- [ ] 第一关玄关托盘相关物体不悬空
- [ ] 家具脚贴地（沙发、桌子、柜子等）
- [ ] 目标区光圈贴地或贴容器表面

## HUD 布局

- [ ] 1440px 宽度下所有面板不重叠
- [ ] 1280px 宽度下仍然可用
- [ ] Tab 键可以收起/展开任务面板
- [ ] R 键可以收起/展开事件日志
- [ ] 事件日志默认收起
- [ ] H 键可以隐藏全部辅助 UI
- [ ] 小地图可以收起和展开
- [ ] 音效开关在得分面板内可用
- [ ] 重新开始按钮不遮挡小地图
- [ ] 刷新页面后 UI 状态保持（localStorage）

## 小地图

- [ ] 第一关默认能看全客厅、卧室、玄关
- [ ] 第二关默认能看全餐厅、厨房
- [ ] 小地图可以拖动（pan）

- [ ] 房间名中文显示
