# 人工 Golden Path 检查清单

> 自动 E2E 测试已覆盖 command→store→UI 流程，但空间寻路、门洞穿越、碰撞、拾取交互等物理/3D 行为需人工验证。
> 本清单中所有项目均为 **未验证**，自动 E2E 不得将其标记为已验证。

## 第一关：出门准备（task-leave-home）

### 1. 空间移动与导航

- [ ] WASD 从客厅走到卧室（经过门洞）
- [ ] WASD 从卧室走回客厅（门洞双向可通过）
- [ ] 不撞空气墙（门洞、墙角处可正常通过）
- [ ] 不穿视觉墙（无法穿过墙体模型）
- [ ] 鼠标拖动可正常转视角
- [ ] V 键可切换第一/第三人称视角

### 2. 记忆系统

- [ ] 实际靠近钥匙，按 E 保存记忆
- [ ] 记忆槽中出现钥匙记忆卡片
- [ ] 触发猫事件后，钥匙记忆变为 outdated
- [ ] 记忆 outdated 时有视觉/音效反馈

### 3. 拾取与放置

- [ ] 实际靠近物品，按 F 拾取
- [ ] 拾取后手部/持有状态正确显示
- [ ] 实际打开床头柜（按 F），取得手机
- [ ] 容器打开/关闭动画正常
- [ ] 关闭的容器中物品不可见，打开后可见
- [ ] 把钥匙、手机、雨伞送到玄关托盘并放置
- [ ] 物品放置后在托盘中可见，不悬空、不嵌入

### 4. 脚本事件

- [ ] 猫事件自然触发（钥匙位置变化）
- [ ] 手机铃声事件触发
- [ ] 事件只触发一次，不会重复触发

### 5. 完成流程

- [ ] 三件物品全部放入玄关托盘后，关卡完成
- [ ] 完成后自动进入 Probe（记忆测试）
- [ ] Probe 完成后进入 Result（结果页）
- [ ] 结果页显示评级、重玩按钮

### 6. 连续通关稳定性

- [ ] 连续完整通关 1 次
- [ ] 连续完整通关 2 次
- [ ] 连续完整通关 3 次
- [ ] 连续完整通关 4 次
- [ ] 连续完整通关 5 次
